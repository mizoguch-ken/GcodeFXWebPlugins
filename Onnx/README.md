## Overview 概要
 * Use ONNX Runtime to infer deep learning.
 ONNX Runtime を使用して、ディープラーニングを推測します。

## Usage 使い方
 * Run with Javascript  
 Javascriptで実行する  

 e.g.  

```
onnx.load('path/to/mnist.onnx');
var data = {'Input name': [...]}
onnx.run(data);
```

- Input Info

```
onnx.load('path/to/mnist.onnx');
getInputInfo();
```
